## Electron

### インストールしておく

```sh
npm install electron -g
```

### 実行する

- IntelliJ IDEA で Node.js プラグインを有効にする
- 実行設定に Node.js から Electron 追加する
- 開発時は環境変数に `TEXT_EMOJI_GENERATOR_ENV=development` を追加する
